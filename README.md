# NSP Sponsor Pages

Public, static sponsorship pages for Negative Split Productions owned events. Served on Vercel at `sponsor.mynegativesplit.com`.

- `houston-resolution-run/` -- Houston Resolution Run sponsorship details + pricing

Each `index.html` is a self-contained build artifact (fonts via CDN, images embedded). The source lives in the private NSP ops repo and is regenerated there. Do not hand-edit these files; update the source and re-publish.
